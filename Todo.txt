
--------------------------------------------------
  Importance élevée
--------------------------------------------------

- Listes de lecture dynamiques :
  - Chargement des listes. (OK)
  - Modification d'une liste.
  - Mise à jour des morceaux. (à terminer)
  - Vérifier que les champs ne sont pas vides.
  - Si un multi-critères ne contient qu'un enfant, on peut remonter celui-ci au niveau supérieur.
  - Trier les types de critères par ordre alphabétique.
  - Ajouter des unités pour certains types (taille du fichier, débit, etc.).

- Lire les métadonnées d'un morceau au chargement. (à terminer)
  - MP3 : lire les tags APE.
  - FLAC.
  - Ogg Vorbis.

- Écrire les métadonnées d'un morceau.
  - Ne pas oublier de mettre à jour la taille du fichier.

- Boite de dialogue Edit Songs. (à terminer)

- Menu contextuel dans les listes : ajouter le/les morceaux à une liste.

- Implémenter le glisser-déposer au sein d'une liste statique.

- Menu contextuel pour les listes de lecture.
  - Supprimer une liste de lecture statique.
  - Supprimer une liste de lecture dynamique.
  - Créer un dossier.

- Implémenter la lecture aléatoire.

- Import depuis iTunes / SongBird.

- Tester le scrobbling Last.fm.

  - Ajouter une option pour tronquer les noms (40 caractères pour iTunes).
