
--------------------------------------------------
  Importance élevée
--------------------------------------------------

- Listes de lecture dynamiques :
  - Vérifier que les champs ne sont pas vides.
  - Si un multi-critères ne contient qu'un enfant, on peut remonter celui-ci au niveau supérieur.
  - Trier les types de critères par ordre alphabétique.
  - Vérifier qu'il n'y a pas de boucles pour le critère "Liste de lecture".

- Import depuis iTunes / SongBird.
  - Importer les morceaux.
  - Importer les listes statiques.
  - Importer les listes dynamiques.
  - Importer les dossiers.
  - Gestion des conflits (fichier XML, métadonnées, fichier déjà ajouté...).
    - Importer l'information "Skip when shuffling".
    - Note, Coché, SkipShuffle, Compilation : Ne pas changer, Toujours changer, Changer si défini.
    - Lectures : Ne pas changer, Toujours changer, Fusionner (les nombres de lecture sont additionnés, et la date de dernière lecture est la plus récente).

- Tester le scrobbling Last.fm.

  Importance moyenne
--------------------------------------------------

- Boite de dialogue Edit Song(s) :
  - Gérer l'option "Ajustement du volume".

- Gérer les dossiers de listes de lecture.
  - Suppression d'un dossier : pouvoir garder ou supprimer le contenu.
  - Implémenter le glisser-déposer pour déplacer les listes de lecture et les dossiers.
  - Pouvoir créer une liste directement dans un dossier.

- Implémenter le glisser-déposer pour créer une nouvelle liste de lecture.
- Griser les boutons indisponibles suivant l'état (lecture, pause, etc.).
- Implémenter correctement la répétition (répétition de la liste, du morceau, etc.).

- Optimisation : exécuter les requêtes SQL dans un thread séparé.
  - Utiliser des transactions.

- Organiser automatiquement les fichiers.
  - Pouvoir préciser le schéma du nom ("%ARTIST%/%ALBUM%/{{%DISC%-}%TRACK% }%TITLE%")
  - Ajouter une option pour tronquer les noms (40 caractères pour iTunes).

- Barre d'état :
  - Pouvoir afficher des notifications (Last.fm, ajout de fichiers, etc.).
  - Afficher les informations sur les morceaux sélectionnés.
